Jailbreak Tweak

SpeedSpring

加快 SpringBoard 系统动画速度，App 启动、切换、开合文件夹更跟手。

功能说明

  • 默认把系统动画倍速调到 1.8x，操作反馈更快
  • 速度可在 1.0x - 5.0x 之间自定义，1.0x 即恢复系统默认
  • 设备过热（达到 serious/critical）或开启低电量模式时，自动把倍速收回 1.0x，不跟系统的省电意图对着干
  • 改配置后无需重启 SpringBoard，执行一条 notifyutil 命令即可热更新

使用方法

  • defaults write /var/mobile/Library/Preferences/com.biefan.speedspring.plist speed -float 2.0
  • defaults write /var/mobile/Library/Preferences/com.biefan.speedspring.plist enabled -bool false（关闭）
  • notifyutil -p com.biefan.speedspring/reload（使配置立即生效）

兼容信息

包标识：com.biefan.speedspring

适配系统：iOS 15.0 及以上（无根越狱，如 Dopamine / roothide）

依赖说明：ElleKit
