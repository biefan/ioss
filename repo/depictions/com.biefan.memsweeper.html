Jailbreak Tweak

MemSweeper

系统内存告警时，自动清理白名单之外的后台 App，释放内存。

功能说明

  • 监听系统内存压力事件（warn / critical），触发时才清理，不做无意义的定时轮询
  • 默认白名单：SpringBoard、电话、信息，避免误杀常驻通信 App
  • 白名单可自定义追加，写入偏好文件后热更新，无需重启 SpringBoard

使用方法

  • defaults write /var/mobile/Library/Preferences/com.biefan.memsweeper.plist whitelist -array "com.your.app"
  • defaults write /var/mobile/Library/Preferences/com.biefan.memsweeper.plist enabled -bool false（关闭）
  • notifyutil -p com.biefan.memsweeper/reload（使配置立即生效）

兼容信息

包标识：com.biefan.memsweeper

适配系统：iOS 15.0 及以上（无根越狱，如 Dopamine / roothide）

依赖说明：ElleKit；用到 FrontBoard 私有框架，系统大版本升级后如失效需要适配
