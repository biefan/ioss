Jailbreak Tweak

StorageSweep

后台定期清理各 App 沙盒内的缓存目录，释放存储空间。

功能说明

  • 通过 LaunchDaemon 每 24 小时触发一次，清理超过 N 天未修改的缓存文件
  • 只清理各 App 自己的 Library/Caches 目录，不触碰系统目录、不需要额外权限
  • 不依赖 MobileSubstrate/ElleKit，不注入任何进程，纯脚本 + 系统 launchd

使用方法

  • defaults write /var/mobile/Library/Preferences/com.biefan.storagesweep.plist olderThanDays -int 3
  • defaults write /var/mobile/Library/Preferences/com.biefan.storagesweep.plist enabled -bool false（关闭）

兼容信息

包标识：com.biefan.storagesweep

适配系统：iOS 15.0 及以上（无根越狱，如 Dopamine / roothide）

依赖说明：无（不依赖 ElleKit / mobilesubstrate）
