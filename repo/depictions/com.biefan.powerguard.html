Jailbreak Tweak

PowerGuard

低电量时自动开启系统低电量模式，过热或低电量模式下清理后台 App，减少 CPU/GPU 负载。

功能说明

  • iOS 不对越狱插件开放 CPU 调频接口，这个插件不做、也做不到直接限制 CPU 最高频率
  • 真正的省电手段是自动开关系统自带的「低电量模式」——iOS 官方的 CPU/GPU 降频、后台限制机制，跟你在设置里手动开的是同一个开关，只是自动化了：电量低于阈值且没充电时自动开，充电或电量回升后自动关
  • 另外在设备过热（serious/critical）或低电量模式已经开着（不管是不是这个插件开的）时，顺手清理一遍白名单之外的后台 App，5 分钟内最多清理一次，避免频繁打断使用
  • 不依赖其他插件，单独安装即可生效；跟 SpeedSpring 搭配使用效果更好（后者过热时会自动把动画倍速收回 1.0x）

使用方法

  • defaults write /var/mobile/Library/Preferences/com.biefan.powerguard.plist lowBatteryThreshold -int 20（低于 20% 自动开低电量模式，默认值）
  • defaults write /var/mobile/Library/Preferences/com.biefan.powerguard.plist restoreBatteryThreshold -int 80（充到 80% 或插上充电自动关，默认值）
  • defaults write /var/mobile/Library/Preferences/com.biefan.powerguard.plist autoLPM -bool false（只保留后台清理，关掉自动低电量模式）
  • defaults write /var/mobile/Library/Preferences/com.biefan.powerguard.plist whitelist -array "com.your.app"
  • defaults write /var/mobile/Library/Preferences/com.biefan.powerguard.plist enabled -bool false（整体关闭）
  • notifyutil -p com.biefan.powerguard/reload（使配置立即生效）

兼容信息

包标识：com.biefan.powerguard

适配系统：iOS 15.0 及以上（无根越狱，如 Dopamine / roothide）

依赖说明：ElleKit；用到 FrontBoard、LowPowerMode 私有框架，系统大版本升级后如失效需要适配
