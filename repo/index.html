Jailbreak Source

My Jailbreak Repo

这是你的个人越狱插件源首页。先修改 repo.conf，再把这里的标题、介绍和源地址换成你自己的信息。

Sileo / Zebra / Cydia 添加源时使用你的公开 URL，例如：https://example.com/repo/

添加源

将你的仓库目录部署到公开静态站点后，把源地址添加到包管理器即可。

放置插件

把构建好的 .deb 文件放入 repo/debs/，然后重新生成索引。

介绍页

每个插件建议单独准备一个 HTML 介绍页，放在 repo/depictions/。

维护流程

  • 编辑 repo.conf 设置仓库元信息
  • 上传或复制新的 .deb 文件到 repo/debs/
  • 执行 ./scripts/build-repo.sh 生成索引
  • 把整个 repo/ 目录同步到静态托管环境
